About

Saya adalah lulusan dari Universitas Pasundan Bandung Fakultas Teknik, Jurusan Teknik Informatika konsentrasi Sistem Informasi dengan raihan Ipk 3.23. Saya lulus tahun 2017. Selama berkuliah, saya aktif dalam beberapa kegiatan organisasi dikampus dan juga aktif mengikuti komunitas-komunitas sekitaran Bandung. Saya seorang yang cepat belajar, memiliki kemampuan analisis yang cukup baik, attention to detail, dan saya sangat suka berproses. Saya memiliki pengalaman dan kemampuan di bidang perancangan sistem juga memiliki ketertarikan dalam desain aplikasi. Saya terbiasa menggunakan tools desain seperti photoshop, Illustrator, dan Adobe XD.

Saya memiliki pengalaman kerja kurang dari 1 tahun. Saya pernah menjadi seorang Appplication Trainer disalah satu start up di Bandung. Tugas Saya yaitu memastikan user untuk dapat menggunakan aplikasi yang dibangun dengan baik dan juga melakukan user acceptance test. Saya menjadi bagian dalam pembangunan aplikasi sistem informasi dalam project ini. Selain itu saya juga berpengalaman sebagai IT helpdesk dan administration staff di salah satu perusahaan otomotif di Indonesia.
